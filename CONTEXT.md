# skills

A public repo of personal agent skills. This context covers vocabulary introduced by the `content` skills (`/design-system`, `/slides`) for building reveal.js presentations.

## Language

**Design system**:
A reveal.js theme (CSS custom properties for color/typography), a set of named layouts, and brand assets, paired with a human-readable `brand-guidelines.md`. Bootstrapped by `/design-system`, consumed by `/slides`. A repo may hold multiple, stored under `design-systems/<name>/`.
_Avoid_: brand kit, style guide

**Layout**:
A named, reusable slide template within a design system (e.g. title, section-header, bullets, image-with-caption, quote) that `/slides` composes into a deck.
_Avoid_: template, slide type

**Deck**:
A single reveal.js presentation generated by `/slides`, rendered against one design system, living under `slides/<deck-name>/`.
_Avoid_: presentation, slideshow

**brand-guidelines.md**:
The human-readable half of a design system. Its frontmatter carries a one-line `description`, shown by `/slides`'s picker when no design system is specified; the body documents the brand for humans.
_Avoid_: style guide (as a filename), brand doc
